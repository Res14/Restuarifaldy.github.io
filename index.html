Tentang Saya

Saya adalah IT Support Specialist yang berfokus pada keandalan infrastruktur bisnis ritel. Karir saya dibangun di atas pemecahan masalah nyata di lapangan—mulai dari menangani error aplikasi POS, merakit dan memperbaiki PC Kasir, hingga mengonfigurasi jaringan dan CCTV toko.

Tidak hanya terpaku pada perbaikan teknis harian, saya memiliki *passion* kuat terhadap Otomatisasi dan Efisiensi IT. Saya mengembangkan sistem *monitoring* menggunakan script Python dan membangun *dashboard* berbasis web dengan PHP Laravel untuk mengontrol ratusan cabang secara terpusat, memastikan bisnis berjalan tanpa gangguan (Zero Downtime).
